📊 Adventure Works Sales Dashboard (Power BI)
🚀 Project Overview

This project is an end-to-end business intelligence dashboard built using Power BI on the Adventure Works dataset.
The dashboard provides comprehensive insights into sales performance, cost analysis, reseller distribution, employee workload, and product trends to support data-driven decision-making.

🎯 Objectives
Analyse sales vs cost trends to identify profitability patterns
Track monthly, quarterly, and yearly performance
Evaluate top-performing products and categories
Monitor progress towards sales targets ($100M goal)
Understand reseller distribution across regions and types
Assess employee workload balance
Analyse geographical sales and quantity distribution
Enable interactive drill-down insights and navigation
📌 Key Features
📈 Sales & Cost Analysis
Monthly comparison of Sales vs Cost
Identification of profitable periods and trends
